# Sisyphus 模式

> 永不放弃。500 步，无限修复，预算耗尽自动续命。

## 启用

```
/sisyphus    ← 开启/关闭 (跨重启持久化)
```

## 参数对比

| 参数 | 普通模式 | Sisyphus 模式 |
|---|---|---|
| 步数预算 | 50 | **500** |
| 修复次数 | 3 | **无限 (∞)** |
| 预算耗尽 | 停止 | **自动续命 (最多3轮 = 1500步)** |
| 状态栏 | `⛩️化神境` | `🔄⛩️化神境` |

## 适用场景

- 大型重构（跨多文件、多模块）
- 框架迁移（如 React Class → Hooks）
- 从零搭建项目（架构 + 实现 + 测试）
- 疑难 Bug 追踪（需要反复验证）

## 停止条件

1. 所有 todo 项标记完成
2. 用户输入 `/stop`
3. 1500 步全部消耗（500步 × 3轮自动续命）

## 持久化

Sisyphus 状态持久化到 `.daoismcode/` 目录，跨重启保持。重启后自动恢复。
