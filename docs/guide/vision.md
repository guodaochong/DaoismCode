# 多模态视觉

> `/paste` 一键粘贴截图，自动路由到多模态视觉模型。

## 用法

```
1. Win+Shift+S 截图 (进入剪贴板)
2. 在 DaoismCode 中输入 /paste
3. Agent 自动切换到金丹境 (多模态视觉模型)
4. 分析图片 → 返回诊断 + 修复
```

## 能力矩阵

| 场景 | 示例 |
|---|---|
| 🔍 **错误诊断** | 截图错误弹窗 → 分析原因 → 生成修复 |
| 🎨 **UI转代码** | 截图设计稿 → 生成 HTML/CSS/React |
| 🐛 **Bug诊断** | 截图布局错误 → 定位 CSS 问题 |
| 📊 **图表分析** | 截图图表 → 提取数据和趋势 |
| 🏗️ **架构图理解** | 截图架构图 → 理解组件关系 |

## 技术实现

`/paste` 读取系统剪贴板图片，Base64 编码后通过 `analyze_image` 工具发送给多模态模型。路由器自动检测图像输入并切换到金丹境。

手动使用：

```
用户: "分析 error.png 这个报错"
→ Agent 调用 analyze_image({ path: "error.png", prompt: "分析..." })
```
