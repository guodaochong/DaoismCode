# 并行子智能体

> 自动检测独立任务，一键启动最多 4 个并行子 Agent。快 58%。

## 自动检测

Agent 实时分析你的消息，检测独立任务：

```
你: "帮我实现注册、登录、密码重置三个功能"
         ↓ 自动检测
         ✓ 3 个独立任务动词
         ✓ 无依赖关键词 (然后/之后/基于)
         ↓
         🚀 3 个子智能体同时启动
```

## 检测模式

| 模式 | 示例 | 触发 |
|---|---|---|
| 数字列表 | `1. 注册 2. 登录 3. 重置` | ✅ |
| 分号列表 | `实现注册; 添加登录; 创建重置` | ✅ |
| 顿号列表 (3+) | `实现注册、添加登录、创建重置` | ✅ |
| 多动词 | `实现注册 和 添加登录` | ✅ |
| 顺序依赖 | `先修复bug，然后写测试` | ❌ |
| 单一任务 | `修复这个bug` | ❌ |

## 执行

```
🚀 3个并行子智能体在 12.3秒内完成 (3/3 成功)

✅ 任务1: 实现用户注册功能 (5步)
✅ 任务2: 实现用户登录功能 (4步)
✅ 任务3: 实现密码重置功能 (6步)
```

## 安全机制

| 机制 | 作用 |
|---|---|
| **权限锁** | 串行化权限提示，防止 UI 死锁 |
| **120s 超时** | 每个子智能体 2 分钟后自动终止 |
| **深度限制** | 子智能体不能再生成子智能体 (max depth 2) |
| **文件冲突警告** | 系统提示禁止并行任务触碰同一文件 |

## 速度对比

```
串行: Task1(8s) → Task2(6s) → Task3(10s) = 24秒
并行: max(8s, 6s, 10s)                    = 10秒   ← 快58%
```
