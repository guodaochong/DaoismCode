# 命令列表

> 23 个斜杠命令。在 TUI 输入框中输入 `/` 触发。

## 核心命令

| 命令 | 说明 |
|---|---|
| `/paste` | **粘贴剪贴板截图 → 自动分析（多模态视觉模型）** |
| `/sisyphus` | **开启/关闭 Sisyphus 模式（500步+无限修复+自动续命）** |
| `/test <file>` | **自动生成单元测试 + 自动修复 + 运行验证** |
| `/review` | **六维代码审查（bug/安全/性能/风格/破坏性/测试）** |
| `/scan` | **全代码库扫描（安全/bug/性能/技术债/缺失测试）** |
| `/index` | **构建语义搜索索引** |
| `/index --force` | **强制全量重建索引** |
| `/search <query>` | **语义代码搜索 — 按含义搜索** |

## 模型与路由

| 命令 | 说明 |
|---|---|
| `/model` | 境界选择器（↑↓选择 筑基/金丹/元婴/化神） |
| `/model 1\|2\|3\|4` | 快速切换到 筑基/金丹/元婴/化神 |
| `/model auto` | 开启智能路由 |
| `/routing` | 路由统计（各境界消耗 + 建议） |

## 会话管理

| 命令 | 说明 |
|---|---|
| `/compact` | 压缩对话上下文 |
| `/sessions` | 列出已保存会话 |
| `/resume <id>` | 恢复之前的会话（todo 自动恢复） |
| `/clear` `/new` | 清空对话，重新开始 |
| `/undo` `/redo` | 撤销/重做上一轮 |

## 其他

| 命令 | 说明 |
|---|---|
| `/init` | 自动生成 `Daoism.md`（项目扫描器，25+ 语言） |
| `/hooks` | 显示已配置的钩子 |
| `/mcp` | 显示已连接的 MCP 服务器 |
| `/stop` | 中断当前 Agent 步骤 |
| `/help` | 显示可用命令 |
| `/exit` | 退出 |
