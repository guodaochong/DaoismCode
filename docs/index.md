## 快速开始

::: code-group

```bash [安装]
git clone https://github.com/jumpingbirds/DaoismCode-AI.git
cd DaoismCode-AI
pnpm install
```

```bash [配置]
# 任何 OpenAI 兼容端点即可
echo '{
  "provider": "自定义",
  "baseURL": "https://your-api/v1",
  "apiKey": "YOUR-KEY",
  "model": "your-model",
  "enableRouting": true
}' > .daoismcode/config.json
```

```bash [启动]
pnpm tui          # 启动 TUI
pnpm build:exe    # 构建独立 .exe → dist/daoism.exe (103MB)
```

:::
